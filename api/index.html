    Introduction

    Welcome to Wandify’s API!

    Seamlessly integrate with our platform to empower your applications and uncover valuable insights. Whether you’re enhancing contact information, exploring organizational details, or optimizing sales and recruitment processes, our API provides a flexible and comprehensive solution tailored to meet your diverse needs.

    Authentication

    Wandify uses API keys to allow access to the API. You may request an API key here.

    Wandify expects the API key to be included in all API requests to the server as a query string parameter.

    Example request

    curl --location 'https://api.wandify.io/api/v1/health?apiKey=YOUR_API_KEY' \
--header 'content-type: text/plain'

    Example request

    GET https://api.wandify.io/api/v1/health

    If authentication is successful, you should expect to see status 200

    Rate limits

    Requests to our API will be limited by different type of credits in accordance with your current pricing plan.

    Rate limits

    For each endpoint, the limit is 5 requests per second.

    Account information retrieval

    Example request

    curl --location 'https://api.wandify.io/api/v1/accountInfo?apiKey=YOUR_API_KEY' \
--header 'content-type: text/plain'

    Example response

    {
  "firstName": "Rosetta",
  "lastName": "Wiza",
  "accountStatus": "active",
  "paidTill": 1748610716429,
  "contactsCredits": 46641,
  "exportCredits": 358577,
  "role": "ROLE_ADMIN",
  "notLocked": true,
  "notRemoved": true
}

    This endpoint retrieves detailed account information based on the provided API key. It returns the user’s personal details, account status, subscription information, and available contact and export credits.

    HTTP Request

    GET https://api.wandify.io/api/v1/accountInfo

    Query parameters

    Parameter Description
    apiKey Your generated API key

    Contact enrichment

    Example request

    curl --location -X POST 'https://api.wandify.io/api/v1/searchContacts' \
-H 'content-type: application/json' \
-d '{ "apiKey": "a538be94c8eb4e34a2df809d2d20d868", "linkedinUrl": "https://linkedin.com/in/username", "firstName": "Rosetta", "lastName": "Wiza", "companyName": "Wandify Inc.", "companyDomain": "wandify.io", "email": "example@domain.com" }'

    Example response

    [
  {
    "type":"EMAIL",
    "value":"example@domain.com"
  },
  {
    "type":"PHONE",
    "value":"1234567890"
  }
]

    This endpoint enriches a profile’s contact information using various parameters provided, such as LinkedIn URL, first name, last name, company name, company domain, and email. The input only accepts LinkedIn regular URLs, not Sales Navigator or Talent / Recruiter LinkedIn products. It will deduct 1 contact credit for each non-empty record returned.

    HTTP Request

    POST https://api.wandify.io/api/v1/searchContacts

    JSON Body Parameters

    Parameter Description Example
    apiKey Your generated API key a538be43c8eb4w34a2dfsf9d2d202368
    linkedinUrl (optional) Profile URL from LinkedIn https://linkedin.com/in/username
    firstName (optional) The person's first name Rosetta
    lastName (optional) The person's last name Wiza
    companyName (optional) The person's company name Wandify Inc.
    companyDomain (optional) The person's company domain wandify.io
    email (optional) The person's email example@domain.com
